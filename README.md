# Hotel-Information-Agent
An AI-powered agent that answers guest questions for a hotel.

## Project Overview
Hotel-Information-Agent is a portfolio project built in **Azure AI Foundry**. It behaves like a simple virtual concierge that responds to common guest questions (amenities, restaurant hours, and check-out times) using a friendly, professional tone.

## Features
- Answering questions about amenities  
- Answering questions about restaurant hours  
- Answering questions about check out times  

## Architecture and Technologies
- Azure AI Foundry  
- AI model: gpt-5-mini  
- Instructions: You are a helpful hotel concierge attending to guest inquiries and concerns. Be nice and have a great attitude to the customer.
